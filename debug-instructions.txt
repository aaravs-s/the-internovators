When you want to test SafeWalkers on your own computer, follow these instructions.

First time setup:
  - Install git at https://git-scm.com/install/ and make sure it is installed by
    running "git -v" in your terminal or command line.
  - Clone the repository:
    "git clone https://github.com/liamb09/the-internovators.git"
  - Go into the project folder:
    "cd the-internovators"
  - Create a Python virtual environment:
    "python -m venv .venv"
  - Activate it on Windows:
    ".venv\Scripts\Activate.ps1"
  - Activate it on Mac/Linux:
    "source .venv/bin/activate"
  - Install the project dependencies:
    "pip install -e ."

To run the site:
  - Activate the virtual environment if it is not already active.
  - Run:
    "fastapi dev main.py"
  - Open the local URL it prints, usually http://127.0.0.1:8000.
  - Press CTRL+C in the terminal when you want to stop the site.

Project notes:
  - The root main.py creates and launches the app.
  - Templates live in templates.
  - Static CSS and JavaScript live in static.
  - Prototype JSON data lives in data.
